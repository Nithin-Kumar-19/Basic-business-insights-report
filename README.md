 # Sales Data Analysis with Python

A simple project to analyze sales data using Python pandas and matplotlib. This script processes sales CSV files, finds top products, analyzes monthly sales trends, and generates a basic business insights report with visualizations.

## Features

- Load sales data from CSV or Excel files.
- Clean and preprocess data for analysis.
- Aggregate sales data by product and month.
- Visualize top products and monthly sales trends.
- Print a summary business insights report to console.
